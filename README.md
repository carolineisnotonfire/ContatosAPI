# ContatosAPI
Uma API rápida de contatos em C# usando como banco de dados o PostgreSql, apenas para um pequeno estudo sobre utilização do banco juntamente com a linguagem em .NET 6

# Contexto
Tava afim de fazer uma API pequena de uma só model pra ter como base pra se eu quiser fazer outras e também dar uma aprofundadinha em conceitos do novo, espetacular, GIGANTESCO .net 6, aí me veio a ideia de fazer com Postgres porque eu to com o banco aqui parado e queria aprender faz um tempo como integrar e etc.
